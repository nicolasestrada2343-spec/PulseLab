# PulseLab

## Overview

PulseLab is an educational platform for Electrocardiogram (ECG) simulation and analysis developed using Java, Spring Boot, SQL persistence, and Artificial Intelligence integration.

The system allows students and instructors to simulate cardiac rhythms, analyze ECG signals, manage patient records, and receive educational recommendations through an AI assistant restricted to the cardiology domain.

## Technologies Used

* Java 21
* Spring Boot
* Spring Data JPA
* H2 Database
* Thymeleaf
* Groq API (LLM Integration)
* Maven

## Main Features

* Real-time ECG simulation
* Patient management
* Medical staff management
* SQL persistence
* Educational AI assistant
* Cardiac rhythm analysis

## Project Structure

* Model Layer
* Repository Layer
* Service Layer
* Controller Layer
* View Layer
